# Skill: The Content Multiplier (`multiply`)
### *Detailed Workflow for AITechDad Content Atomization*

When this skill is triggered, follow these sequential steps to "atomize" a core piece of content (like a script or transcript).

## 🎯 Goal
Turn 1 high-value asset into a comprehensive distribution bundle that feels native to every platform.

## 🛠️ Execution Steps

### Step 1: Context Absorption
Analyze the source file for:
- The "Big Idea" (The core transformation).
- The "Spicy Take" (The unique angle that gets clicks).
- Key data points or metaphors used.

### Step 2: The Multi-Platform Bundle
Generate the following assets exactly as described:

**1. The LinkedIn "Thought Leader" Post**
- **Structure**: Hook (first 2 lines) -> The "Problem" -> The "Aha!" realization -> 3 Bullet points of value -> Closing CTA.
- **Tone**: Professional but punchy. No "corporate filler."

**2. The X (Twitter) Thread**
- **Count**: 8-10 tweets.
- **Tweet 1**: A scroll-stopping hook with an "open loop."
- **Tweets 2-9**: Step-by-step breakdown. Use short, high-energy sentences.
- **Tweet 10**: Viral loop CTA (Like/Retweet/Follow).

**3. The Short-Form Video Hooks (x5)**
- Give me 5 different ways to start a 60-second video about this topic. 
- Variations: The "Controversial" start, The "How-to" start, The "I can't believe" start.

### Step 3: Brand Tone Audit
Before outputting, run a final check based on the voice in `CLAUDE.md`.
- Replace "Utilize" with "Use."
- Replace "Strategic implementation" with "Building a system."
- Ensure one dad-joke or metaphor is present in the LinkedIn post.

## 🏁 Output Format
Organize everything into a clean markdown report with clear headers for each platform.

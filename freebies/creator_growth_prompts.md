## 🚀 Phase 1: The "Viral Spark" (Ideation)

### 1. Niche Trend Hunter
```
Search my recent content library. Based on [TOPIC], what are 3 "counter-intuitive" angles I haven't covered yet? Make them spicy but valuable.
```

### 2. The Hook Laboratory
```
I have this idea: [IDEA]. Write 7 scroll-stopping hooks. Give me 2 that are curiosity-driven, 2 that are "how-to" bold, and 1 that is a total hot take. 
```

### 3. "Explain Like I'm 5" (For Viral Clips)
```
Take this complex AI concept: [CONCEPT]. Break it down so a 5th grader gets it, but keep the "cool factor" for my tech-savvy audience.
```

### 4. YouTube Thumbnail Brainstormer
```
Based on my script [PASTE SCRIPT], give me 5 high-CTR thumbnail concepts. Describe the visual and the "big text" on the image.
```

### 5. Newsletter Subject Line Split-Tester
```
Write 5 subject lines for this week's issue. Give me one "Classic AITechDad," one mysterious, and one that is a direct benefit.
```

## ✍️ Phase 2: The "Production Powerhouse" (Scripting)

### 6. The "Quick-Start" Video Script
```
Write a 60-second TikTok/Reel script about [TOPIC]. 0-3s: Hook. 3-15s: The "Aha!" moment. 15-50s: The meat. 50-60s: Soft CTA.
```

### 7. Newsletter Draft (Deep Dive)
```
Turn these messy notes into a structured newsletter: [NOTES]. Use my "Dad-joke-adjacent but tech-smart" voice. Include an intro, 3 main points, and a "Tool of the Week."
```

### 8. The "No-Fluff" LinkedIn post
```
Take this blog post [PASTE] and turn it into a punchy LinkedIn post. No corporate speak. Just "here's the problem, here's the fix, let's go."
```

### 9. X/Twitter Thread Composer
```
Turn [TOPIC/ARTICLE] into a 10-tweet thread. Tweet 1: The Hook. Tweets 2-9: The Value (Step-by-Step). Tweet 10: The CTA + Viral Loop.
```

### 10. Carousel Scripting
```
Draft a 7-slide Instagram carousel script. Slide 1 is the hook. Slide 7 is the save/share CTA.
```
